What This Demo Shows

The left panel shows a programmatically generated skeleton — the logical structure (facts, rules, disjunctions, proof chain) is deterministic and provably correct. The right panel shows the LLM's attempt to re-express this skeleton as natural language with formal anchors. The LLM produces a context, question, entity/predicate mappings, and a structured completion with <step> tags. Validation status indicates whether the output passed format checks (JSON parse + <think>/<step>/<logic> tags present).

10/10 fully valid completions with an 8B base model. A larger instruct model (32B+) is expected to achieve >60% pass rate. Invalid outputs are still shown for diagnostic purposes.
